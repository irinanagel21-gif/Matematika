Matematika

Минимальный рабочий продукт для проекта «Математика простым языком». Страница, структура и базовые стили готовы.

Узнать больше

Что внутри

  • Страница `index.html`
  • Стили в `assets/css/styles.css`
  • JavaScript в `assets/js/script.js`
  • Картинка в `assets/img/image.png`

Цель MVP

Показать основную идею продукта в виде одной рабочей страницы и подготовить структуру для дальнейшего развития.

Структура проекта

Matematika построена по простой архитектуре, чтобы быстро перейти от идеи к первой рабочей версии.

  1. index.html — основная страница
  2. assets/css/styles.css — стили
  3. assets/js/script.js — скрипт
  4. assets/img/image.png — изображение
