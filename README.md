# Matematika

Matematika — минимальный рабочий продукт для проекта «Математика простым языком».

## Что это
Это простая демонстрационная веб-страница, которая показывает идею MVP: одну страницу с описанием продукта, базовым оформлением и иллюстрацией.

## Что внутри
- `index.html` — основная страница проекта.
- `assets/css/styles.css` — стили оформления.
- `assets/js/script.js` — файл для будущей логики и интерактивных элементов.
- `assets/img/image.png` — иллюстрация или картинка проекта.
- `.gitignore` — правила, которые не позволяют коммитить служебные или временные файлы.

## Как открыть
Откройте файл `Matematika/index.html` в браузере.

## Как запускать локально
1. Откройте терминал.
2. Перейдите в папку проекта:
   ```bash
   cd ~/Desktop/Matematika
   ```
3. Можно просто открыть `index.html` в браузере двойным кликом.
4. Если хотите, подключите простой локальный сервер:
   ```bash
   python3 -m http.server 8000
   ```
5. Затем откройте в браузере:
   ```
   http://localhost:8000
   ```

## Целевая аудитория
Проект подходит для:
- тех, кто хочет быстро показать идею обучения математике в виде одного веб-продукта;
- репетиторов, которые хотят демонстрировать ученикам концепцию онлайн-сервиса;
- владельцев небольших стартапов, которым важна простая и понятная MVP-страница.

## Что можно улучшить дальше
1. Добавить реальные тексты и описания продукта.
2. Сделать навигацию между разделами.
3. Подключить форму обратной связи или кнопку «Начать».
4. Разделить CSS на несколько файлов, если интерфейс станет больше.

## Почему это полезно
Такая структура позволяет быстро показать идею, сохранить файл в Git и развивать проект без лишнего мусора.
