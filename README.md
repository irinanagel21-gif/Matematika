# Matematika

Matematika — минимальный рабочий продукт для проекта «Математика простым языком».

## О проекте
Это небольшая одностраничная демонстрация, которая показывает структуру MVP для образовательного веб-проекта.

## Что внутри
- `index.html` — основная страница.
- `assets/css/styles.css` — оформление и адаптивные стили.
- `assets/js/script.js` — файл для будущей логики.
- `assets/img/image.svg` — иллюстрация.
- `.gitignore` — список файлов, не сохраняемых в Git.

## Как запускать локально
1. Открой терминал.
2. Перейди в папку проекта:
   ```bash
   cd ~/Desktop/Matematika
   ```
3. Открой `index.html` двойным кликом или запусти локальный сервер:
   ```bash
   python3 -m http.server 8000
   ```
4. Перейди в браузере:
   ```text
   http://localhost:8000
   ```

## Что можно улучшить дальше
1. Добавить реальные тексты, примеры и описание курса.
2. Сделать навигацию по разделам.
3. Подключить форму обратной связи или кнопку «Начать».
4. Расширить стили и добавить адаптивность для мобильных экранов.

## Почему это полезно
Такая простая структура позволяет быстро показать идею, хранить проект в Git и развивать его без лишних файлов.
